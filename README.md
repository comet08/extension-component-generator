# Vscode Extension Component Generator

폴더 + 우클릭 후 Generate Styled Component 명렁어 클릭
컴포넌트명 입력(PascalCase)

출력:
kebab-case
  ㄴ Component.tsx
  ㄴ Component.styled.tsx
  ㄴ index.ts
